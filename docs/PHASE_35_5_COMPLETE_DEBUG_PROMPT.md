## 🚀 ЯК ТЕСТУВАТИ

### 1. Задеплоїти на хостинг

```bash
npm run build
# Потім задеплоїти dist/ на Netlify/Vercel/інший хостинг
```

### 2. Відкрити в Telegram

1. Відкрити Telegram Mini App
2. Відкрити DevTools (Telegram Desktop: Ctrl+Shift+I, або мобільний браузер)
3. Дивитись на консоль

### 3. Що шукати в консолі

При успішному завантаженні ви побачите:
```
=== 📄 INDEX.HTML ЗАВАНТАЖЕНО ===
=== 🚀 MAIN.TSX ЗАВАНТАЖЕНО ===
=== 🌍 ПЕРЕВІРКА ГЛОБАЛЬНИХ ОБ'ЄКТІВ ===
=== 🔧 ПЕРЕВІРКА КОНСТРУКТОРІВ ===
=== 📱 TELEGRAM WEBAPP ===
```

При помилці ви побачите:
```
=== 🔴 HTML ПОМИЛКА ===
Повідомлення: Object is not a constructor (evaluating 'new jr')
Файл: https://...
Рядок: ...
Колонка: ...
Stack: ...
```

### 4. Повідомити результат

Скопіюйте з консолі:
- Повідомлення про помилку
- Stack trace
- Все, що починається з `===`
